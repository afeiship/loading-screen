# loading-screen
> Pure loading-screen for spa.

## usage
```html
<!--1. add styles -->
<link rel="stylesheet" type="text/css" href="//unpkg.com/@feizheng/loading-screen@1.0.2/dist/style.css"/>

<!--2. add html template -->
<section class="loading-screen">
  <div class="spinner">
    <div class="rect1"></div>
    <div class="rect2"></div>
    <div class="rect3"></div>
    <div class="rect4"></div>
    <div class="rect5"></div>
  </div>
  <span>应用加载中</span>
</section>
```
